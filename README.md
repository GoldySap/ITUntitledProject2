# ITUntitledProject2

## Om spillet

ITUntitledProject2 er et hacker-/whistleblower-spill hvor spilleren infiltrerer et korrupt datasystem for å avdekke hemmelige dokumenter og sannheten bak et mektig selskap.

Spillet foregår hovedsakelig gjennom et terminalgrensesnitt inspirert av gamle operativsystemer og hacking-verktøy. Spilleren må navigere filer, hacke systemer, dekryptere data og unngå å bli oppdaget.

Temaet for prosjektet er **"Breaking the System"**, hvor målet er å utfordre kontroll, overvåkning og skjulte maktstrukturer.

---

## Funksjoner
- Terminal-basert gameplay
- Simulert hacking
- Filnavigasjon og kommandoer
- Historiedrevet opplevelse
- Spenning og tidsbasert risiko
- Laget i Unity
